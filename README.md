# 📝 Todo App

A simple and responsive Todo List web application built using **HTML, CSS, and JavaScript**.  
This project allows users to add, mark complete, and delete tasks to manage their daily activities efficiently.

🔗 Live Demo:
https://todo-app-azure-tau.vercel.app/
---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- ⌨️ Enter key support for quick task addition
- 💾 Local Storage support to save tasks permanently
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
